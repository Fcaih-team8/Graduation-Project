# AI-NIDS-Project
A network intrusion detection system (NIDS) that utilizes machine learning and AI techniques to detect various types of attacks in real-time across network, application, and web environments.
## Objectives
- Develop a network intrusion detection system using AI.
- Provide real-time alerts for different types of cyber-attacks.
- Ensure compatibility across different environments (Network, Application, Web).
## Technologies
- Python
- Fast/Django for Backend
- TensorFlow for AI
- React for Frontend
- Wireshark for Network Monitoring

## Dataset
The project will use publicly available datasets like CICIDS and UNSW-NB15, which include various types of attacks (DDoS, DoS, port scanning, malware, etc.) for training and testing the machine learning models.
